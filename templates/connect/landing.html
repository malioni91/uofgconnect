UofG Connect

UofG Connect’s primary objective is to help students find their friends’ location around the campus and provide an intuitive interface to keep them in touch. Students will be able to register only through their university e-mail address and once logged in, they could search for their friends. The app will then display the friends’ location, and allow the user to filter the list of displayed friends by their name.

User Registration & Authentication

A user registration and authentication process allows each individual user-student to register and access the student’s availability map in the campus.

Extract User Location

The application requires access to the user’s current location in order to track the user’s presence in the campus. However, the presence is not persistent after closing the application.

Google Maps Display

A Google Map is included in the main page, capable of displaying data and popup markers containing user locations.

Interaction Between Students

The application enables user interaction between the students of the campus such as short invitations.
